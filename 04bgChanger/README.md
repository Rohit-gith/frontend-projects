# 🎨 Background Color Changer

A fun and interactive React app that changes background color with a single click.

## ✨ Features
- 🎨 Multiple color options
- 🖱️ One-click color change
- 📱 Fully responsive design
- ⚡ Smooth transitions

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Vite
