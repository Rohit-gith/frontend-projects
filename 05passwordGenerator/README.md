# 🔐 Password Generator

A fast and responsive password generator built with React.js that creates strong, secure passwords with custom options.

## ✨ Features
- 🔢 Custom password length (6-100 characters)
- 🔤 Include/exclude numbers & special symbols
- 📋 One-click copy to clipboard
- ⚡ Built with Vite for fast performance

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Vite
