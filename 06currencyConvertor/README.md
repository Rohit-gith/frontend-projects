# 💱 Currency Converter

A real-time currency converter built with React.js that converts between multiple world currencies instantly.

## ✨ Features
- 🌍 Support for multiple world currencies
- 🔄 Swap currencies with one click
- ⚡ Real-time exchange rates via API
- 📱 Fully responsive design

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Vite
- Currency Exchange API
